# Network File System (NFS) — Free Edition

Plugin gratuito de código abierto para aaPanel que permite gestionar NFS (Network File System) de forma gráfica: crear exportaciones, montar directorios remotos, monitorear el servidor y diagnosticar fallos automáticamente.
